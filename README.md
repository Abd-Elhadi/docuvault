# DocuVault 📁

**AI-Powered Document Management Platform**

DocuVault is a full-stack, production-grade document management system that allows users to securely upload, organize, search, and share documents, with AI-powered text extraction and intelligent categorization.

## Features

### Authentication & Security

- JWT-based authentication using **httpOnly cookies**
- Secure file access via **AWS S3 presigned URLs**
- Role-based and permission-aware document sharing
- Expiring share links
